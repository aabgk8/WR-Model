# Fantasy PPG WR Model

This repo contains the WR college-to-NFL fantasy modeling pipeline and the current draft-class prediction work.

## Main folders

- `wr_data/wr_data/`: active scripts, raw inputs, measurable data, and rebuildable pipeline intermediates

## Key docs

- `wr_data/wr_data/README.md`: pipeline overview and active file layout
- `wr_data/wr_data/MODEL_RESULTS_SUMMARY.md`: latest model variants, metrics, and top 15 WR boards
